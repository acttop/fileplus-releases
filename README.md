# FilePlus

쌓여있는 파일, 흩어진 이름들 — 파일 탐색·중복 탐지·전문 검색·메타정보 일괄 편집·자동 분류·이름 변경까지 한 화면에서 처리하는 Windows용 데스크톱 파일 관리 도구입니다.

## 다운로드

**👉 [다운로드 페이지 열기](https://acttop.github.io/fileplus-releases/)**

계정(로그인) 없이 누구나 위 페이지에서 최신 버전을 받을 수 있습니다.

## 주요 기능

- 파일 탐색 · 검색 · 복사
- PDF / DOCX / 이미지 메타정보 일괄 편집
- SQLite 기반 전문(全文) 검색
- SHA-256 기반 중복 파일 탐지
- 규칙 기반 자동 분류 + Excel 보고서 출력
- 폴더/파일 이름 일괄 변경

## 안내

- 설치 프로그램이 아닌 **포터블(portable) 실행파일**입니다. 다운로드 후 바로 실행하세요.
- 코드 서명 인증서가 없어 Windows에서 "알 수 없는 게시자" 경고가 뜰 수 있습니다. **추가 정보 → 실행**을 눌러 진행해주세요.
- 이 저장소는 배포(실행파일 제공) 전용이며, 소스 코드는 별도로 관리됩니다.
