# FilePlus

폴더 하나에 파일이 수백, 수천 개씩 쌓여 있는데 뭐가 뭔지 못 찾겠는 경험, 다들 한 번쯤 있으실 거예요. 같은 파일이 이름만 바뀐 채 세 번, 네 번 중복돼 있고, 문서 제목이나 작성자 정보는 죄다 "제목 없음"이고, 특정 키워드가 들어간 파일을 찾으려면 하나하나 열어봐야 하는 그 답답함.

**FilePlus**는 이런 문제를 해결하려고 만든 Windows용 데스크톱 파일 관리 도구입니다. 설치 없이 실행파일 하나로 바로 쓸 수 있고, 여러 개의 반복적인 파일 정리 작업을 한 화면에서 처리할 수 있게 설계했어요.

## 뭘 할 수 있나요

**파일 탐색 · 검색 · 복사**
드라이브 전체를 빠르게 탐색하고, 조건에 맞는 파일을 검색해서 원하는 위치로 바로 복사할 수 있어요.

**메타정보 일괄 편집**
PDF, DOCX, 이미지 파일의 제목·작성자·키워드 같은 메타정보를, 파일을 하나씩 열지 않고 한 화면에서 일괄로 수정합니다.

**전문(全文) 검색**
PDF·DOCX·TXT 파일 내용을 SQLite에 인덱싱해두고, 파일명이 아니라 **내용 안의 키워드**로 검색할 수 있어요.

**중복 파일 탐지**
SHA-256 해시 비교로 완전히 똑같은 파일을 찾아내고, 이름이 비슷한 파일들도 클러스터링해서 보여줍니다.

**규칙 기반 자동 분류**
확장자·날짜·파일명 패턴 규칙을 정해두면 자동으로 분류해주고, 분류 결과는 Excel 보고서로 바로 뽑을 수 있습니다.

**이름 일괄 변경**
폴더/파일 이름을 하나씩 또는 규칙에 따라 한 번에 바꿀 수 있어요.

## 왜 만들었나

문서가 많은 환경일수록 "정리"는 미루기 쉬운 일이 되기 마련이에요. 도구가 흩어져 있으면(중복 찾기 따로, 이름 바꾸기 따로, 메타정보 편집 따로) 정리 자체를 시작하기가 부담스러워지거든요. FilePlus는 이런 반복 작업들을 하나의 앱 안에 모아서, 파일 정리를 "큰 맘 먹고 하는 일"이 아니라 "가볍게 켜서 하는 일"로 만들고 싶었습니다.

## 안내

- 설치 프로그램이 아닌 **포터블(portable) 실행파일**입니다. 다운로드 후 바로 실행하세요.
- 코드 서명 인증서가 없어 Windows에서 "알 수 없는 게시자" 경고가 뜰 수 있습니다. **추가 정보 → 실행**을 눌러 진행해주세요.
- 이 저장소는 배포(실행파일 제공) 전용이며, 소스 코드는 별도로 관리됩니다.
